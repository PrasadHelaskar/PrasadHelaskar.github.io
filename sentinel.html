🛡️ Sentinel

Sentinel is a lightweight automation intelligence layer designed to monitor CI pipelines, analyze execution logs, detect failures, and deliver actionable insights in real time.

GitHub Repo Back to Portfolio

⚠️ Problem

Modern CI/CD pipelines generate huge amounts of logs, but teams often struggle to quickly understand:

  • Which test failed
  • Why it failed
  • What caused the issue
  • Where debugging should start

💡 Solution

Sentinel acts as an observability layer on top of automation systems by converting raw logs into structured, actionable insights.

⚙️ Key Features

🚀 Rule-based failure detection
📊 Structured log analysis
🔔 Slack notifications
📦 JSON-based run storage
🧩 Extensible analyzer design
⚡ CLI integration

🏗️ Architecture

CI Pipeline (GitHub Actions / Jenkins)
            ↓
      Test Execution (Pytest)
            ↓
        Log Generation
            ↓
        Sentinel CLI
            ↓
   Rule-Based Analyzer Engine
            ↓
 Structured Result (JSON Output)
            ↓
 Slack Notifications + Run Storage

🔧 Tech Stack

Python Pytest GitHub Actions Slack Webhooks JSON Storage Logging Framework
